Here is where you create and edit Workflows, the management areas for content items. The default Workflows are Media, Story and Template, but you can add new Workflows or change the existing ones to fit your site's needs.

Search
Searches Workflows by name. Do a blank search to list all Workflows or click a letter for an alphabetical list.
Add a New Workflow
Click this link to create a new Workflow. It brings you to the Workflow Profile page.
Existing Workflows
Click Edit to modify a Workflow's profile. Click Log to view its event history. To delete a Workflow, check the Delete box, scroll to the bottom of the page and click the Delete Checked button.
