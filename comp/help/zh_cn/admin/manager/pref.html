Here you can adjust settings and change the way certain information is displayed. Search for the Preference you want to edit, or do a blank search to view all Preferences.

Bricolage Instance Name
Label used in window titles and the Welcome message on the login page. Click Edit to get to the Preference Profile page, edit the name, and click Save.
Date/Time Format
How the date and time are displayed. Click Edit to get to the Preference Profile page, select the desired format, and click Save.
Default Asset Sort
The default Story/Media field to sort on. Click Edit to get to the Preference Profile page, select the desired field, and click Save.
Default Search
The default search, Simple or Advanced, in Find Media and Find Stories. Click Edit to get to the Preference Profile page, select the desired sort type, and click Save.
List Name Format
How people's names (such as those of users and contributors) are displayed in lists. Click Edit to get to the Preference Profile page, select the desired format, and click Save.
Name Format
How people's names are displayed outside of lists. Click Edit to get to the Preference Profile page, select the desired format, and click Save.
Permissions Widget
The style of widget used in controlling Permissions for users. Click Edit to get to the Preference Profile page, click the circle next to the desired format, and click Save.
Search Results / Page
The number of rows displayed per page in searches. Click Edit to get to the Preference Profile page, select the desired number of pages, or Off to disable paging, and click Save.
Time Zone
The time zone your Bricolage recognizes. Click Edit to get to the Preference Profile page, select the desired time zone, and click Save.
