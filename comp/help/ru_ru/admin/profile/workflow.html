Свойства (Properties)
Задайте имя Рабочего Процесса и его краткое описание. Затем выберите для него базовый тип: Текст (Story), Медиа (Media) или Профиль (Profile).
Исходный стол (Start Desk)
Выберите стол, на который чаще всего будут помещаться новые разделы контента в данном Рабочем Процессе. Наименование этого стола будет появляться первым в выпадающих меню для Рабочих Процессов. Чтобы создать стол нажмите на клавишу «Новый стол» (New Desk).

Для перехода к следующей странице Профиля нажмите клавишу «Далее (Next)».

Добавить новый стол (Add a New Desk)
Данная ссылка используется для включения нового стола в список столов Рабочего Процесса. По ней вы попадаете в Профиль Рабочего Стола, где выберете Стол, который вы хотите добавить или создадите новый, нажав клавишу «Новый стол».
Столы (Desks)
Для того, чтобы изменить Исходный Стол поставьте отметку рядом с желаемым столом. Нажмите «Редактировать» для модификации профиля Стола. Нажмите «Журнал» для просмотра списка выполненных действий. Для удаления Стола из Рабочего Процесса поставьте отметку в окошке «Удалить».

По завершении нажмите клавишу «Сохранить» (Save).
