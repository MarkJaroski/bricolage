Эта форма предназначена для создания и редактирования информации о Местах Размещения (Destinations), контента, а также информации о том как эти места должны быть использованы в процессе публикации.

На первом этапе создания нового Места Размещения только первые две секции формы являются видимыми. После их заполнения нажмите ‘Далее’ для перехода к остальной части формы. При редактировании существующего Места Размещения заполненная форма высвечивается целиком.

Свойства (Properties)

Имя (Name)
Это наименование Места. Оно должно быть уникальным, но используется только для того, чтобы легче было отличить одно Место от другого.
Описание (Description)
Это наименование Места. Оно должно быть уникальным, но используется только для того, чтобы легче было отличить одно Место от другого.
Метод пересылки (Move Method)
Метод, используемый для копирования файлов на Место. Множество доступных методов в значительной степени зависит от того, как сконфигурирован установленный БРИКОЛАЖ. Но в любом случае должны быть доступны следующие методы:
Файловая система (File System)
Простое копирование в рамках локальной файловой системы.
FTP
Загрузка на удаленный сервер по протоколу FTP.
SFTP
Загрузка на удаленный сервер по защищенному протоколу FTP.
WebDAV
Загрузка на удаленный сервер по протоколу HTTP с использованием WebDAV.
Копирование ресурсов (Copy Resources)
Если эта отметка поставлена, то ресурсы, предназначенные к загрузке на сервер, будут предварительно скопированы во временный каталог.
Публикация (Publishes)
Если эта отметка установлена (она установлена по умолчанию), то это Место должно быть использовано для публикации подготовленных материалов.
Просмотр (Previews)
Если эта отметка установлена, то Место должно быть использовано для предварительного просмотра подготовленных к публикации материалов.
Сайт (Site)
Если вы не определили никаких сайтов, то здесь по умолчанию устанавливается значение «Default Site» (сайт по умолчанию). Иначе просто выберите нужный сайт из списка сайтов.

Каналы Вывода (Output Channels)

В этой секции указываются Каналы Вывода, для которых будет использоваться данное Место Размещения.

Действия (Actions)

Данная часть формы используется для задания, редактирования или удаления действий. Действия соответствуют различным типам обработки данных, выполняемой в процессе их пересылки на Место.

Для общего случая задано только одно действие, типа ‘Move’ (переместить), которое выполняет текущую операцию копирования или загрузки.

Серверы (Servers)

Данная часть формы используется для задания, редактирования или удаления серверов. Здесь конкретизируется способ пересылки материалов на сервер в процессе их публикации. Например, если для копирования данных используется протокол FTP, то на сервер должны быть переданы также имя пользователя и пароль.
