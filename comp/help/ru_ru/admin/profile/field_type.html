Свойства (Properties)
Введите краткое описание Поля и его максимальную длину (если необходимо). Отметки «требуется» и «повторяется» ставятся также по необходимости.
Атрибуты отображения (Display Attributes)
Этот список атрибутов будет изменяться в зависимости от типа Поля, но обязательно будет включать следующие атрибуты:
Разрешить несколько (Allow multiple)
Применяется только для выделения Полей и обозначает возможность одновременного выделения нескольких из них. Например, при помощи нажатой клавиши Shift. (select)
Заголовок (Caption)
То же, что и метка (Label), но для поля даты. Содержит календарную дату. (date)
Колонки (Columns)
Число колонок в области текста. (textarea)
Значение по умолчанию (Default Value)
Значение по умолчанию для данного Поля. (textfield, radio, pulldown, select, textarea)
Метка группы (Group Label)
То же, что и метка (Label), но для Полей, соответствующих точечным отметкам. (radio)
Метка (Label)
Метка, которую Администратор Элементов видит в списке Полей. (textfield, radio, checkbox, pulldown, select, textarea)
Максимальный размер (Maximum Size)
Максимальный размер данного Поля. (textfield, textarea)
Опции, Метка (Options, Label)
Это список строк, содержащих метки и значения. Заметьте, что «Опции, Метка» отображает формат каждой строки. «Опции» – это значимый атрибут элемента <option>, например, в списке <select<, с «Меткой» показываемой пользователю. (radio, pulldown, select)
Строки (Rows)
Число строк в области текста. (textarea)
Размер (Size)
Размер данного Поля. (textfield, select)

Если вы хотите удалить это Поле из Элемента, то поставьте отметку «Удалить этот профиль» (Delete this Profile).

По завершении нажмите клавишу Сохранить. Для отмены изменений нажмите Отмена.
