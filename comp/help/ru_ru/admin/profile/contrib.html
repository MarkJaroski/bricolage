Свойства (Properties)
Здесь задаются полное имя автора, а также префикс (Mr/Mrs) и/или суффикс (например, Jr) если необходимо.
Профиль Автора (Contributor Profile)
Воспользуйтесь выпадающим меню для выбора типа Автора.

Нажмите клавишу Далее для перехода на следующую страницу.

Контакты (Contacts)
Здесь задается контактная информация Автора. Воспользуйтесь выпадающими меню для уточнения средств связи (телефон, факс, е-мейл и др.), а также пустыми полями справа для ввода данных. Если четырех полей мало, то нажмите Добавить еще.
Стандартные атрибуты роли (Default Role Attributes)
Здесь задается краткое описание роли Автора.

По завершении ввода нажмите клавишу Сохранить.
