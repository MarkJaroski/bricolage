Свойства (Properties)
Задайте имя Группы, а также краткое описание ее роли.
Тип Группы (Group Type)
Для выбора Типа Группы воспользуйтесь выпадающим меню.

Нажмите Далее для перехода к следующей странице Профиля.

Члены (Members)
Определите, кто из зарегистрированных пользователей войдет в данную Группу. Выбирайте по одному из списка слева и нажимайте клавишу «Включить в список». В списке справа отображаются все текущие члены Группы. Для удаления кого-либо выберите его имя в списке и нажмите клавишу «Удалить из списка».
Разрешения (Permissions)
Нажмите эту клавишу для перехода на страницу разрешений Группы.

По завершении нажмите клавишу Сохранить.
