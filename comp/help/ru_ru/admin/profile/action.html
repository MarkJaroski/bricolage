Эта форма используется для того, чтобы добавлять или редактировать Действия (Actions), которые различаются по типам обработки данных в процессе публикации.

Свойства (Properties)

Тип (Type)
Тип Действия, которое будет выполнено. Наиболее часто используется «Переместить», которое и выполняет реальную пересылку или загрузку выходных данных на сервер.
Очередность (Order)
Используется для установки очередности, в которой будут выполняться Действия в процессе публикации.
