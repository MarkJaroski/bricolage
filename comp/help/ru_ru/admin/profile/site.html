Эта форма предназначена для создания или редактирования Сайтов (site).

Свойства (Properties)
Введите имя Сайта, имя домена и, по желанию, краткое описание.
Членство в Группах (Group Memberships)
Определите какие привилегии сайт должен иметь. Выберите из списка слева Группы, в которые сайт будет включен. Выбирайте по одной группе и нажимайте клавишу «Добавить в список» (Add to List). В списке справа показаны все Группы, которым уже принадлежит сайт. Для удаления такой группы выберите ее имя и нажмите клавишу «Удалить из списка» (Remove from List).

По завершении нажмите клавишу «Сохранить».
