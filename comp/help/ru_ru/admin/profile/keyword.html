Эта форма используется для добавления или редактирования ключевых слов, которые связываются с материалами или категориями.

Свойства (Properties)

Имя ключевого слова (Keyword Name)
Это необходимое поле.
Экранное имя ключевого слова (Keyword Screen Name)
Определяет каким образом данное имя будет отображаться на экране. (Например, ключевое слово = 'George', Экранное имя ключевого слова = 'George Washington'). Если значение screen_name не указано явно, то в этом качестве используется Имя ключевого слова.
Имя порядка сортировки (Sort Order Name)
Слово, используемое для сортировки ключевых слов. Если не задано явно, то в этом качестве используется Имя ключевого слова.
