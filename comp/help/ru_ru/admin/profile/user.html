Имя (Name)
В этих полях задайте полное имя пользователя, а также необходимые префикс или суффикс.
Контакты (Contacts)
Здесь задается контактная информация пользователя. Воспользуйтесь выпадающими меню для выбора средств связи (телефон, е-мейл, факс и т.п.), а также поля справа для ввода данных. Если необходимо использовать более четырех полей, то нажмите клавишу «Добавить еще».
Членство в группах (Group Memberships)
Здесь определяются привилегии, которые будет иметь пользователь. Из списка, расположенного слева, выберите те административные группы, в которые пользователь будет включен. Выбирайте группы по одной и добавляйте в список нажатием клавиши «Добавить в список». В списке справа отображаются все группы, в которые пользователь уже включен. Для удаления какой-либо из них выберите ее в этом списке и нажмите клавишу «Удалить из списка».
Логин и Пароль (Login and Password)
Задайте стартовый Логин и Пароль пользователя. Позднее пользователь сможет изменить эти значения.

По завершении нажмите клавишу «Сохранить» (Save)
