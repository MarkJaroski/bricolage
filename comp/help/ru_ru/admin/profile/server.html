Сервер server определяет конкретное место для копирования или загрузки публикуемых материалов.

Свойства (Properties)

Имя хоста (Host Name)
TИмя хоста или IP-адрес сервера. Если необходимо, задайте адрес конкретного порта.
ОС (OS)
Операционная система, которая функционирует на сервере. Этим определяется способ формирования имен файлов на сервере.
Корневой каталог для документов (Document Root)
Полный путь на сервере к корневому каталогу для документов. Сюда будут копироваться публикуемые материалы.
Логин (Login)
Имя пользователя для захода на сервер (при использовании FTP или SFTP).
Пароль (Password)
Пароль для входа на сервер.
Куки (Cookie)
HTTP куки, которое используется некоторыми «методами пересылки данных» (такими как WebDAV) для аутентификации себя серверу.
Активный (Active)
Сервер может использоваться только в том случае, когда он отмечен как «активный» (active).
