Вы можете указать, чтобы оповещения отправлялись либо отдельному польователю, либо группе пользователей.

Групповые получатели (Group Recipients)
Определяет, что это оповещение отправляется группе или группам пользователей. Выберите группу в списке слева, «Доступные Группы», а затем нажмите клавишу «Добавить в Список». Группа будет перемещена в список «Текущие Группы», который находится справа. Для удаления группы из списка текущих групп нажмите клавишу «Удалить из Списка».
Индивидуальные получатели (User Recipients)
Определяет, что это оповещение отправляется индивидуальному пользователю, либо нескольким пользователям. Выберите пользователя в списке слева, «Доступные Пользователи», а затем нажмите клавишу «Добавить в Список». Пользователь будет перемещен в список «Текущие Пользователи», который находится справа. Для удаления пользователя из списка текущих пользователей нажмите клавишу «Удалить из Списка».

По завершении нажмите клавишу «Сохранить» либо «Отмена» (для отмены внесенных изменений).
