Свойства (Properties)
Введите имя Элемента, имя ключа и краткое описание. Имя ключа представляет собой уникальный идентификатор, состоящий из одного слова. Выберите тип "прожигателя", который будет использоваться вашими шаблонами. При добавлении нового Элемента необходимо также задать его Тип.
Членство в группах (Group Memberships)
Членство в группах определяет уровень доступа пользователей при чтении, редактировании или создании Элементов. Выберите группу в списке «Доступные Группы», а зетем нажмите клавишу «Добавить в список». Тем самым, выбранная группа будет добавлена в список «Текущие группы». Для удаления группы выберите е в списке «Текущие группы» и нажмите клавишу «Удалить из списка». При создании нового Элемента нажмите клавишу «Далее». Вы перейдете на страницу с полным Профилем Элемента.
Сайты (Sites)
Элементы высшего уровня (такие как Текст или Медиа) могут использоваться совместно несколькими сайтами. Эта секция недоступна для элементов низших уровней. В секции «Сайты» вы можете подключать или отключать сайты от Элемента. Вы также можете указать Первичный Канал Вывода для каждого сайта.
Каналы Вывода (Output Channels)
Элементы высшего уровня ассоциируются с одним или несколькими Каналами Вывода. Эта секция недоступна для элементов низших уровней. Каналы Вывода определяют как ваши тексты будут отформатированы или куда ваши тексты и медиа данные будут отправлены. У них разные шаблоны, поэтому вы можете выводить одни и те же Элементы различными способами. Необходимо, чтобы с каждым Элементом высшего уровня был ассоциирован как минимум один Канал Вывода.
Существующие Субэлементы (Existing Subelements)
Фактически Элементы являются контейнерами, содержащими другие элементы и поля данных. Для того, чтобы добавить Субэлемент в Элемент, нажмите клавишу «Добавить Субэлемент». Появится диалоговая форма «Выберите Субэлементы». Поле поиска используется стандартным образом. Вы можете просмотреть либо часть, либо весь список доступных Субэлементов. Естественно, что Субэлементом не может быть Элемент высшего уровня. Для добавления Элементов поставьте отметки в соответствующих им окошках «Добавить к Элементу», а затем нажмите клавишу «Добавить Элементы». Если вы решили не добавлять никаких элементов, то нажмите клавишу «Добавить Элементы», не устанавливая никаких отметок. Таким образом вы просто вернетесь на страницу Профиля Элемента.
Поля Пользователя (Custom Fields)
В этих полях хранятся данные. Наряду с субэлементами они определяют структуру Элемента. В данной секции вы можете изменить значение по умолчанию для поля. Можете изменить порядок элементов, выбрав номер во всплывающем меню. Для редактирования поля нажмите ссылку «Редактировать». Вы перейдете к диалоговой форме редактирования Профиля Поля. Здесь вы можете изменить все данные, которые касаются поля, за исключением имени ключа и типа (область текста, отметка и др.) Для удаления поля поставьте отметку в окошке «Удалить», а затем нажмите одну из клавиш «Сохранить» в конце страницы.
Добавить новое Поле (Add New Field)
Здесь вы добавляете новые поля в секцию Полей Пользователя. Выберите тип поля в группе отметок слева. В зависимости от того, что вы выбрали, изменится множество доступных атрибутов, показанное справа. Атрибуты детально описаны в Field Type Profile Help Page

По завершении нажмите клавишу Сохранить или Сохранить и остаться. Нажатие клавиши Возврат вернет вас обратно в Менеджер Элементов без сохранения изменений.
