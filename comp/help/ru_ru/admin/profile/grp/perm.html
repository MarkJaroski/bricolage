Это Профиль Разрешений Группы. Здесь вы определяете разрешения для групп пользователей на доступ к членам групп объектов. Выберите один из типов разрешений в меню «Выберите тип Разрешений».

Для задания разрешений используйте отметки точками (или списки для выбора, если в предпочте для диалоговых форм разрешений будет установлено, как "Select Lists"). Уровни доступа расположены по возрастанию в следующем порядке: Нет/None (по умолчанию), Читать/Read, Редактировать/Edit, Создавать/Create. Заключительное значение «Запретить» (Deny) используется для отмены привилегий пользователей и перекрывает разрешения, предоставленные этим пользователям в других Группах. То есть, если пользователь Вася Иванов является членом Группы, у которой установлено значение «Запретить» для Рабочего Процесса Медиа, то он не сможет читать, создавать или редактировать Медиа, даже если он и является членом другой Группы, которой это разрешено.

Разрешения будут казаться сложными до тех пор, пока вы их не «прочувствуете» (а, может быть, и после этого). Начать лучше всего с чтения раздела АВТОРИЗАЦИЯ в Bric::Security а затем поискать ответы или непосредственно задавать вопросы в список рассылки bricolage-general.
