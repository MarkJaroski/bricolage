Авторы (Contributors) – это люди, создающие контент. Информация о них важна для организации и систематизации контента. Размещение информации об авторах в этой области с последующим присоединением ее к соответствующим разделам контента позволяет:
- контролировать кто, что и когда делает;
- объединять вместе работы одного автора;
- проводить поиск работ конкретного автора.

Поиск (Search)
Поиск Автора по имени. Для получения списка всех Авторов оставьте поле поиска пустым. Возможен также поиск по букве алфавита или по типу.
Добавить нового Автора (Add a New Contributor)
Переход на страничку «Профиль Автора».
Существующие Авторы (Existing Contributors)
Нажмите Редактировать (Edit) для модификации профиля Автора. Нажмите Журнал (Log) для просмотра списка всех выполненных действий. Для удаления профиля Автора поставьте отметку в окошке Удалить (Delete), а затем нажмите клавишу Удалить отмеченные (Delete Checked) в конце страницы.
