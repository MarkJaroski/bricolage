Источник – это некто, ответственный за предоставление того или иного раздела контента. Сайты партнеров, ваш собственный сайт, службы новостей и внештатные сотрудники – все это примеры потенциальных Источников. В начале в системе присутствует только один такой Источник, временно названный «Внутренний Источник». Вы можете его произвольно переименовать, а также добавить и другие источники, от которых вы получаете материалы для формирования контента. Вновь созданный источник может быть немедленно использован при заполнении Профиля раздела контента.

Поиск (Search)
Поиск Источника по имени. Для получения списка всех Источников оставьте поле поиска пустым. Возможен также поиск по букве алфавита.
Добавить новый Источник (Add a New Source)
Переход на страничку «Профиль Источника».
Существующие Источники (Existing Sources)
Нажмите Редактировать (Edit) для модификации профиля Источника. Нажмите Журнал (Log) для просмотра списка всех выполненных действий. Для удаления профиля Источника поставьте отметку в окошке Удалить (Delete), а затем нажмите клавишу Удалить отмеченные (Delete Checked) в конце страницы.
