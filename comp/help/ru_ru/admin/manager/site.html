Здесь вы создаете и редактируете сайты.

Поиск (Search)
Поиск активных сайтов по имени. Для получения списка всех активных сайтов оставьте поле поиска пустым.
Добавить Новый Сайт (Add a New Site)
Нажмите эту ссылку для создания нового сайта. Вы перейдете на страницу Профиля Сайта. Введите требуемую информацию. По завершении нажмите клавишу «Сохранить» в конце страницы.
Существующие Сайты (Existing Sites)
Нажмите Редактировать для модификации Профиля Сайта. Нажмите Журнал для просмотра списка всех действий, выполненных с сайтом. Для удаления сайта поставьте отметку в окошке Удалить, а затем нажмите клавишу Удалить отмеченные в конце страницы.
