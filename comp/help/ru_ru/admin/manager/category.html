«Категория» – это то место на вашем сайте, где публикуется контент. В начале БРИКОЛАЖ содержит единственную категорию «Root Category», но здесь вы можете создавать и редактировать новые Категории. Публикация разделов контента в нескольких Категориях может быть полезной при упорядочивании компонентов вашего сайта.

Поиск (Search)
Поиск Категорий по имени. Для получения полного списка Категорий введите пустое поле поиска. Для получения списка Категорий в алфавитном порядке нажмите на нужную букву.
Добавить новую Категорию (Add a New Category)
Нажмите на эту ссылку для создания новой Категории. БРИКОЛАЖ перейдет на страницу профиля Категории. Для сохранения введенных данных нажмите клавишу «Сохранить» (Save) в конце страницы.
Существующие Категории (Existing Categories)
Нажмите Редактировать (Edit) для модификации профиля Категории. Нажмите Журнал (Log) для просмотра списка всех действий, выполненных с данной Категорией. Для удаления Категории поставьте отметку в окошке Удалить, а затем нажмите в конце страницы клавишу Удалить отмеченные.
