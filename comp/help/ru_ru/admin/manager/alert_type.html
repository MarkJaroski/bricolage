Здесь вы можете задать рассылку вам сообщений или оповещений о выполнении тех или иных действий. Например, если вы отослали новый Текст (Story) для публикации в следующий вторник, то, как только этот срок наступит, БРИКОЛАЖ отправит вам оповещение о том, что действие выполнено.

Поиск (Search)
Поиск активных оповещений по имени. Для просмотра списка всех типов оповещений введите пустое поле поиска.
Добавить новый тип оповещений (Add a New Alert Type)
Нажмите на эту ссылку и вы перейдете на страницу Профиля Типа Оповещений. Введите необходимую информацию, а затем нажмите клавишу «Сохранить».
Существующие Типы Оповещений (Existing Alert Types)
Для изменения Профиля Типа Оповещений нажмите «Редактировать». Для просмотра журнала событий данного типа нажмите «Журнал» (Log). Для удаления Типа оповещений поставьте отметку в окошке «Удалить» рядом с ним и нажмите клавишу «Удалить отмеченные» в конце страницы.

К сведению системных администраторов. В файле bricolage.conf вам необходимо указать SMTP-сервер. В противном случае пользователи не будут получать сообщения по е-мейл.
