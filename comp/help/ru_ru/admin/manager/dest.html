Здесь вы создаете и редактируете Адресатов, которые представляют собой серверы, на которых вы хотите разместить ваш контент. Необходимо отметить, что публикация и пересылка (называемые Действиями) выполняются в раздельных фазах. В процессе публикации контент пропускается через Шаблоны (это называется «прожиганием»), а полученные файлы (как правило в формате HTML) сохраняются в рабочем каталоге. Позднее «монитор размещений», bric_dist_mon, выполнит перемещение «прожженных» файлов из рабочего каталога на сервер Адресата.

Поиск (Search)
Поиск активных Адресатов по имени. Для получения списка всех Авторов оставьте поле поиска пустым.
Добавить нового Адресата (Add a New Destination)
Переход на страничку «Профиль Адресата». Введите необходимую информацию в полях профиля, а затем нажмите клавишу «Сохранить» в конце страницы.
Существующие Адресаты (Existing Destinations)
Нажмите Редактировать (Edit) для модификации профиля Адресата. Нажмите Журнал (Log) для просмотра списка всех действий, выполненных с Адресатом. Для удаления профиля Адресата поставьте отметку в окошке Удалить (Delete), а затем нажмите клавишу Удалить отмеченные (Delete Checked) в конце страницы.
