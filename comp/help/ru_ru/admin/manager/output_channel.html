Каналы Вывода – это места публикации разделов контента. Каналом Вывода, принятым по умолчанию, является World Wide Web. В большинстве случаев он же является и единственным используемым каналом.

Поиск (Search)
Поиск Канала Вывода по имени. Для получения списка всех каналов оставьте поле поиска пустым. Возможен также поиск по букве алфавита.
Добавить новый Канал Вывода (Add a New Output Channel)
Переход на страничку «Профиль Канала Вывода», на которой задаются имя канала и его описание. Для сохранения введенных данных нажмите клавишу «Сохранить» в конце страницы.
Существующие Каналы Вывода (Existing Output Channels)
Нажмите Редактировать (Edit) для модификации профиля Канала вывода. Нажмите Журнал (Log) для просмотра списка всех действий, выполненных с Каналом Вывода. Для удаления Канала Вывода поставьте отметку в окошке Удалить (Delete), а затем нажмите клавишу Удалить отмеченные (Delete Checked) в конце страницы.
