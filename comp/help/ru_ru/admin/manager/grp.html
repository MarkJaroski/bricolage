Разрешения, заданные для Группы, определяют привилегии ее членов. В этой секции вы сможете создавать новые Группы и редактировать разрешения существующих Групп.

Поиск (Search)
Поиск активных Групп по имени. Для получения списка всех Групп оставьте поле поиска пустым. Возможен также поиск Групп по букве алфавита или по типу.
Добавить новую Группу (Add a New Group)
Переход на страничку «Профиль Группы». На этой страничке задаются имя Группы, а также необходимые Тип и разрешения. Для сохранения введенных данных нажмите клавишу «Сохранить» в конце страницы.
Менеджер Групп (Group Manager)
Нажмите Редактировать (Edit) для модификации профиля Группы. Нажмите Журнал (Log) для просмотра списка всех действий, выполненных с Группой. Для удаления Группы поставьте отметку в окошке Удалить (Delete), а затем нажмите клавишу Удалить отмеченные (Delete Checked) в конце страницы.
