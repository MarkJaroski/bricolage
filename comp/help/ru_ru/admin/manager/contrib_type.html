Типы Авторов используются для классификации Авторов по их роли в создании контента. Например, «Писатель» или «Фотограф». Тип Автора используется при работе со страничкой Профиля Автора.

Поиск (Search)
Поиск Типа Автора по имени. Для получения списка всех Типов оставьте поле поиска пустым. Возможен также поиск по букве алфавита.
Добавить новый Тип Автора (Add a New Contributor Type)
Переход на страничку «Профиль Типа Автора». На этой страничке задаются имя, описание и все необходимые атрибуты. Для сохранения введенных данных нажмите клавишу «Сохранить» в конце страницы.
Существующие Типы Авторов (Existing Contributor Type)
Нажмите Редактировать (Edit) для модификации профиля Типа Автора. Нажмите Журнал (Log) для просмотра списка всех выполненных действий. Для удаления профиля Типа Автора поставьте отметку в окошке Удалить (Delete), а затем нажмите клавишу Удалить отмеченные (Delete Checked) в конце страницы.
