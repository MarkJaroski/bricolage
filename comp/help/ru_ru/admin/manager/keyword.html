Здесь вы добавляете, редактируете или удаляете Ключевые Слова.

Поиск
Поиск активных Ключевых слов по имени. Для просмотра списка Элементов в алфавитном порядке нажмите на нужную букву или Все (All).
Добавить новое Ключевое Слово
По этой ссылке выполняется переход на страницу Профиля Ключевого Слова. Введите требуемую информацию, а затем нажмите клавишу Сохранить в конце страницы.
Существующие Ключевые слова
Нажмите Редактировать для того, чтобы изменить Ключевое слово. Нажмите Журнал для просмотра всех действий, выполненных с Ключевым словом. Для удаления Ключевого Слова поставьте отметку в окошке Удалить, а затем нажмите в конце страницы клавишу Удалить отмеченные.
