Здесь вы создаете учетные записи пользователей и устанавливаете для них уровни доступа. Уровень доступа пользователя определяется по его принадлежности к административным группам. Например, если вы хотите, чтобы пользователю Васе Иванову было разрешено редактировать Тексты (Stories), то необходимо зайти на страничку профиля данного пользователя и там включить его в административную группу, которой разрешено выполнять редактирование текстов.

Поиск (Search)
Поиск активных Пользователей по фамилии. Для получения списка всех активных Пользователей оставьте поле поиска пустым.
Добавить нового Пользователя (Add a New User)
Переход на страничку «Профиль Пользователя», на которой задается вся необходимая информация. Для сохранения введенных данных нажмите клавишу «Сохранить» в конце страницы.
Существующие Пользователи (Existing Users)
Нажмите Редактировать (Edit) для модификации профиля Пользователя. Нажмите Журнал (Log) для просмотра списка всех выполненных действий с учетной записью. Для удаления учетной записи Пользователя поставьте отметку в окошке Удалить (Delete), а затем нажмите клавишу Удалить отмеченные (Delete Checked) в конце страницы.
