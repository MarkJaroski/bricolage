Здесь вы создаете и редактируете Рабочие Процессы (области управления для разделов контента). В системе существуют три базовых Рабочих Процесса – это Медиа, Текст и Шаблон. Но вы можете добавлять новые Рабочие Процессы, либо модифицировать существующие по своему усмотрению.

Поиск (Search)
Поиск Рабочих Процессов по имени. Для получения списка всех Рабочих Процессов оставьте поле поиска пустым. Возможен также поиск по первой букве имени.
Добавить новый Рабочий Процесс (Add a New Workflow)
Переход на страничку «Профиль Рабочего Процесса», на которой задается вся необходимая информация. Для сохранения введенных данных нажмите клавишу «Сохранить» в конце страницы.
Существующие Рабочие Процессы (Existing Workflows)
Нажмите Редактировать (Edit) для модификации профиля Рабочего Процесса. Нажмите Журнал (Log) для просмотра списка всех выполненных действий. Для удаления Рабочего Процесса поставьте отметку в окошке Удалить (Delete), а затем нажмите клавишу Удалить отмеченные (Delete Checked) в конце страницы.
