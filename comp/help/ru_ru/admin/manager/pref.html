Здесь вы можете изменить настройки, а также изменить форму представления некоторых данных. Задайте поиск Настройки, которую хотите изменить, или оставьте поле поиска пустым для просмотра всех Настроек.

Имя истемы (Bricolage Instance Name)
Название, используемое в заголовках окон и в приветственном сообщении на странице регистрации. Нажмите «Редактировать (Edit)» для перехода на страницу Профиля Настроек, измените название и нажмите «Сохранить (Save)».
Формат Даты/Времени (Date/Time Format)
Нажмите «Редактировать» для перехода на страничку Профиля Настроек. Выберите желаемый формат представления даты и времени и нажмите клавишу «Сохранить».
Ключ для сортировки материалов (Default Asset Sort)
Поле Текст/Медиа, выбранное по умолчанию для сортировки материалов. Нажмите «Редактировать» для перехода на страницу Профиля Настроек, выберите нужное поле, а затем нажмите клавишу «Сохранить».
Тип Поиска (Default Search)
Тип поиска (простой или расширенный), используемый по умолчанию в режимах «Найти Медиа» и «Найти тексты». Нажмите «Редактировать» для перехода на страницу Профиля Настроек, выберите нужный Тип Поиска, а затем нажмите клавишу «Сохранить».
Формат имени в списке (List Name Format)
Определяет форму представления имен людей в списках (таких как Пользователи или Авторы). Нажмите «Редактировать» для перехода на страницу Профиля Настроек, выберите желаемый формат, а затем нажмите клавишу «Сохранить».
Формат имени (Name Format)
Определяет форму представления имен людей вне списков. Нажмите «Редактировать» для перехода на страницу Профиля Настроек, выберите желаемый формат, а затем нажмите клавишу «Сохранить».
Вид отметок для Разрешений (Permissions Widget)
Возможно использование отметок различных стилей для раздачи Разрешений пользователям. Нажмите клавишу «Редактировать» для перехода на страницу Профиля Предпочтений, нажмите на кружочек рядом с желаемым форматом, а затем нажмите клавишу «Сохранить».
Страница результатов поиска (Search Results/Page)
Число строк, отображаемое на странице результатов поиска. Нажмите «Редактировать» для перехода на страницу Профиля Настроек, задайте необходимое число строк, либо отключите разбиение результатов поиска на страницы. По завершении редактирования нажмите клавишу «Сохранить».
Часовой пояс (Time Zone)
Показывает текущий часовой пояс. Нажмите клавишу «Редактировать» для перехода на страницу Профиля Предпочтений, установите желаемый часовой пояс, а затем нажмите клавишу «Сохранить».
