Здесь вы добавляете, редактируете или удаляете Типы Медиа данных, известные также как MIME типы.

Поиск
Поиск Типов Медиа данных по имени. Для просмотра списка Типов Медиа данных в алфавитном порядке нажмите на нужную букву. Для просмотра списка всех Типов оставьте поле поиска пустым.
Добавить новый Тип Медиа данных
По этой ссылке выполняется переход на страницу Профиля Типа Медиа данных. Введите требуемую информацию, а затем нажмите клавишу Сохранить в конце страницы.
Существующие Типы Медиа данных
Нажмите Редактировать для того, чтобы изменить профиль Типа Медиа данных. Нажмите Журнал для просмотра всех выполненных действий. Для удаления Типа Медиа данных поставьте отметку в окошке Удалить, а затем нажмите в конце страницы клавишу Удалить отмеченные.
