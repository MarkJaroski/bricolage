Этот инструмент применяется для создания Медиа-файлов, которые будут использованы на вашем сайте.

Свойства (Properties)

Заголовок (Title)
Наименование Медиа-материалов. Необходим.
Тип Медиа (Media Type)
Выберите какие Медиа-материалы вы создаете. Необходим.
Источник (Source)
Описание источника, предоставившего данные Медиа-материалы.
Категория (Category)
Место на сайте, где будут опубликованы Медиа-материалы.
Приоритет (Priority)
Показывает степень срочности Медиа. Необходим.
Заглавная дата (Cover Date)
Выберите дату и время, которые будут отображаться с Медиа (это не обязательно дата размещения на сайте, но это та дата публикации, которую увидят читатели).

По завершении нажмите «Создать». После этого вы попадете на страницу Профиля Медиа.
