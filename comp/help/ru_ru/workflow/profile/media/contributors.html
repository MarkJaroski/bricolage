Текущие авторы
Показывает список авторов, которые в настоящее время связаны с этим Медиа-документом. Нажмите ссылку «Роль» для изменения роли Автора. Поставьте отметку в окне «Удалить» и автор будет отсоединен от медиа-документа в момент сохранения изменений.
Поиск
Поиск оставшихся (неассоциированных) авторов по фамилии.
Выберите Авторов
Показывает список авторов, удовлетворяющих условию поиска. Нажмите ссылку «Связать» (Associate) для присоединения автора к медиа-документу.

По завершении нажмите Сохранить.
