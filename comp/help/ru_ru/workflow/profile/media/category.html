Текущая Категория (Current Category)
Показывает категорию, связанную в настоящий момент с данным медиа-документом.
Поиск (Search)
Поиск оставшихся (неассоциированных) категорий по имени.
Выберите Категории (Choose Categories)
Показывает список категорий, соответствующих условию поиска. Нажмите ссылку «Ассициировать» (Associate) для изменения категории медиа-документа.

По завершении нажмите клавишу «Сохранить».
