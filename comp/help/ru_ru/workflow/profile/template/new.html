Используйте этот инструмент для настройки Бриколажа. Здесь вы будете создавать Шаблоны для включения в Профили Текстов и Медиа.

Свойства (Properties)

Элемент (Element)
Выберите из списка Элемент, на котором будет основан данный Шаблон (если таковой есть). Необходим.
Общий (Generic)
Для создания совершенно нового Шаблона поставьте отметку здесь.
Канал вывода (Output Channel)
Цель применения Шаблона. Необходим.
Категория (Category)
Место на сайте, где Шаблон будет применяться. Необходим.
Приоритет (Priority)
Показывает степень срочности данного Шаблона. Необходим.

По завершении нажмите «Создать» (Create) и вы перейдете на страницу Профиля Шаблона.
