1) Информация (Information)

Проследить (Trail)
Показывает все Столы, на которых когда-либо размещался данный Шаблон. (Для нового Шаблона показывает пустой список).
Заметки (Notes)
Нажмите этот значок для того, чтобы присоединить редакторские заметки к Шаблону. Заметки могут быть просмотрены другими пользователями после того как Шаблон отправлен на Стол. Однако они не будут видны при публикации. Если вы видите значок с надписями, то это значит, что какие-то заметки к Шаблону уже присоединены.
Элемент (Element)
Элемент этого Шаблона будет форматировать, если Шаблон является Шаблоном Элемента, выбранным на странице «Создать Новый Шаблон».
Имя Шаблона (Name)
Наименование Шаблона. Оно эквивалентно Типу Материалов если новый Шаблон основывается на существующем Элементе.
Канал Вывода (Output Channel)
Цель (каталог) применения Шаблона.
Категория (Category)
Место на сайте, где Шаблон будет применяться.
URI
Адрес Шаблона на вашем сервере.
Приоритет (Priority)
Показывает степень срочности данного Шаблона. Необходим.
Описание (Description)
Краткое описание Шаблона.
Срок хранения (Expire Date)
Дата и время, когда вы хотите удалить данный Шаблон.

2) Код Шаблона (Template Code)

Элемент (Element)
Элемент, на котором базируется данный Шаблон (если таковой есть).
Элементы данных (Data Elements)
Элементы, которые автоматически появляются в секции Контента данного Шаблона. Те из них, которые помечены как «повторяемые», должны быть добавлены несколько раз. Те элементы, которые помечены как «необходимые», должны быть заполнены пользователем прежде чем Шаблон будет сохранен.
Контейнерные Элементы (Container Elements)
Элементы, которые должны быть включены в Шаблон, приведены в выпадающем списке, размещенном непосредственно под Элементами Данных. Те из них, которые помечены как «повторяемые», должны быть включены несколько раз.

«Сохранить».
