Используйте этот инструмент для создания Текстов для вашего Веб-сайта. Существуют четыре различных типа Текстов, и каждый из них имеет несколько переменных и дополнительных элементов, которые вы можете использовать для управления отображением Текста.

Свойства (Properties)

Заголовок (Title)
Заголовок Текста. Необходим.
Подкаталог (Slug)
Однословный идентификатор, используемый в качестве заключительной части URI данного Текста, а также в качестве подкаталога, в котором этот текст размещается.
Тип Текста (Story Type)
Выберите какой Текст вы создаете. Необходим.
Источник (Source)
Описание источника, предоставившего Текст.
Первичная категория (Primary Category)
Основное место на сайте, где публикуются материалы.
Приоритет (Priority)
Показывает степень срочности Текста. Необходим.
Заглавная дата (Cover Date)
Выберите дату и время, которые будут отображаться с Текстом (это не обязательно дата размещения на сайте, но это та дата публикации, которую увидят читатели).

По завершении нажмите «Создать» (Create). После этого вы попадете на страницу Профиля Текста. На этой странице вы можете вводить данные и добавлять любые элементы, необходимые для оформления Текста. Не забудьте нажать «Сохранить» в конце страницы Профиля Текста для того, чтобы сохранить Текст в вашем Рабочем Пространстве.
