Текущие Авторы
Показывает список авторов, связанных с данным текстом. Нажмите ссылку «Роль» (Role) для того, чтобы изменить роль автора. Поставьте отметку «Удалить» (Delete) и автор будет отсоединен от текста в момент сохранения изменений.
Поиск
Поиск оставшихся (несвязанных) авторов по фамилии.
Выберите Авторов
Показывает список авторов, удовлетворяющих условиям поиска. Нажмите ссылку «Ассоциировать» (Associate) для связывания автора с текстом.

По завершении нажмите клавишу «Сохранить» (Save).
