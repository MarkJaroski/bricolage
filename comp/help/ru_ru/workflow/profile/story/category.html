Текущие Категории
Показывает список категорий, связанных с данным текстом. Нажмите ссылку «Первичная» (Primary) для того, чтобы присвоить категории статус первичной. Поставьте отметку «Удалить» (Delete) и категория будет отсоединена от текста в момент сохранения изменений.
Поиск
Поиск оставшихся (несвязанных) категорий по имени.
Выберите Категории
Показывает список категорий, удовлетворяющих условиям поиска. Нажмите ссылку «Ассоциировать» (Associate) для связывания категории с текстом.

По завершении нажмите клавишу «Сохранить» (Save).
