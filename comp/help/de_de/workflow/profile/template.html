1) Information

Spur
Listet auf, an welche Büros die Vorlage eingecheckt wurde (keine, wenn es sich um eine neue Vorlage handelt).
Notizen
Klicken Sie auf dieses Icon um redaktionelle Notizen an die Vorlage anzufügen. Diese Notizen sind für andere Anwender sichtbar, werden jedoch nicht mit veröffentlicht. Wenn das Icon aussieht, als wäre es beschriftet, sind bereits Notizen mit dieser Vorlage verknüpft.
Güterart
Die Art der Vorlage, wie sie auf der Seite 'Neue Vorlage erstellen' gewählt wurde. Wenn sie mit keinem bestehenden Element assoziiert ist, wird die Art der Vorlage "Generisch" sein.
Name der Vorlage
Der Titel der Vorlage. Diese ist gleich der Güterart wenn die Vorlage auf einem existierenden Element basiert.
Ausgabekanal
Das Ziel für die Vorlage, wenn es einmal ausgebracht ist.
Kategorie
Wo in Ihrer Site die Vorlage ausgebracht wird.
URI
Die Adresse der Vorlage auf dem Server.
Priorität
Teilt Anwendern mit, wie wichtig diese Vorlage ist. Dieses Feld ist zwingend vorgeschrieben.
Beschreibung
Eine kurze Beschreibung der Vorlage.
Verfallsdatum
Datum und Uhrzeit zu der Sie wünschen, daß die Vorlage entfernt wird (verfällt).

2) Vorlagencode

Element
Das Element auf dem die Vorlage basiert, falls zutreffend.
Datenelemente
Elemente, die automatisch im Inhaltsabschnitt der Vorlage erscheinen. Die mit "wiederholbar" gekennzeichneten können mehrmals hinzugefügt werden. Diejenigen, die mit "vorgeschrieben" gekennzeichnet sind müssen vom Benutzer ausgefüllt werden um die Vorlage abspecihern zu können.
Containerelemente
Elemente die der Vorlage hinzugefügt werden können, sie erscheinen in einem Pulldown unter den Datenelementen. Die mit "wiederholbar" gekennzeichneten können mehrmals angefügt werden.

Tragen Sie zusätzlichen Code für die Vorlage in das dafür vorgesehene Feld ein. Klicken Sie auf Speichern wenn Sie fertig sind.
