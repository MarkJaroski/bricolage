Hier erzeugen Sie Benutzerzugänge und definieren Zugangsstufen. Der Zugriff eines Benutzers wird durch seine Mitgliedschaft in administrativen Gruppen bestimmt. Wenn Sie z.B. Heinz Müller erlauben wollen, Beiträge zu editieren, gehen Sie zu seiner Benutzerprofilseite und tragen Sie ihn in eine administrative Gruppe ein, die die Rechte besitzt, Beiträge zu editieren.

Suche
Sucht aktive Benutzer nach Namen. Eine leere Suche listet alle Benutzer auf.
Neuen Benutzer hinzufügen
Klicken Sie hier um einen neuen Benutzer anzulegen. Sie werden zur Benutzerprofilseite weitergeleitet. Tragen Sie die benötigten Informationen ein und klicken Sie am Ende der Seite auf Speichern.
Vorhandene Benutzer
Klicken Sie Edit um das Profil eines Benutzers zu ändern. Klicken Sie Protokoll um die Geschichte der Ereignisse anzusehen. Um ein Benutzerkonto zu löschen, setzen Sie einen Haken in die Löschen-Box, gehen Sie ans Ende der Seite, und klicken Sie auf den Knopf Gewählte Löschen.
