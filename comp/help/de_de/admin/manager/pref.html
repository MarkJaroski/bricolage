Hier können Sie Einstellungen anpassen oder die Art der Darstellung einiger Informationen ändern. Suchen Sie nach den Einstellungen, die Sie verändern möchten, oder führen Sie eine leere Suche durch, um alle Einstellungen anzuzeigen.

Datum-/Zeitformat
Wie Datum und Uhrzeit angezeigt werden. Klicken Sie auf Editieren um auf die Einstellungsprofilseite zu gelangen, wählen Sie das gewünschte Format und klicken Sie dann auf Speichern.
Format der unveränderlichen URIs
Siehe unten, bei URI-Format
Namenslisten-Format
Wie Namen von Personen (z.B. die von Anwendern und Mitarbeitern) in Listen dargestellt werden. Klicken Sie auf Editieren um auf die Einstellungsprofilseite zu gelangen, wählen Sie das gewünschte Format und klicken Sie dann auf Speichern.
Namens-Format
Wie Namen von Personen außerhalb von Listen dargestellt werden. Klicken Sie auf Editieren um auf die Einstellungsprofilseite zu gelangen, wählen Sie das gewünschte Format und klicken Sie dann auf Speichern.
Rechteschalter
Art des Schalters, der für die Kontrolle der Anwenderrechte verwendet wird. Klicken Sie auf Editieren um auf die Einstellungsprofilseite zu gelangen, wählen Sie den Kreis neben dem gewünschten Format und klicken Sie dann auf Speichern.
Zeitzone
Zeitzone, in der sich die Instanz von Bricolage befindet. Klicken Sie auf Editieren um auf die Einstellungsprofilseite zu gelangen, wählen Sie die gewünschte Zeitzone und klicken Sie dann auf Speichern.
