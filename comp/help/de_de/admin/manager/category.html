Eine Kategorie ist der Ort auf Ihrer Website wo die Inhaltsbestandteile veröffentlicht werden. Anfangs gibt es in Bricolage nur eine Kategorie, die Root-Kategorie, aber Sie können in diesem Bereich neue Kategorien erstellen und editieren. Inhalte in mehreren Kategorien zu veröffentlichen kann dabei helfen, die Komponenten Ihrer Website zu ordnen.

Suche
Sucht Kategorien nach Namen. Eine leere Suche listet alle Kategorien auf, über die Buchstaben erhäten Sie eine alphabetische Liste.
Neue Kategorie hinzufügen
Klicken Sie hier um eine neue Kategorie anzulegen. Sie werden zur Kategorienprofilseite weitergeleitet.
Vorhandene Kategorien
Klicken Sie Edit um das Profil einer Kategorie zu ändern. Klicken Sie Protokoll um die Geschichte der Ereignisse anzusehen. Um eine Kategorie zu löschen, setzen Sie einen Haken in die Löschen-Box, gehen Sie ans Ende der Seite, und klicken Sie auf den Knopf Gewählte Löschen.
