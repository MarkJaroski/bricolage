Mitarbeitertypen klassifizieren Mitarbeiter nach Rollen, z.B. "Author" und "Fotograf." Ist ein Mitarbeitertyp definiert, kann ein Anwender es in der Mitarbeiterprofilseite auswählen.

Suche
Sucht Mitarbeitertypen nach Namen. Eine leere Suche listet alle Mitarbeitertypen auf, über die Buchstaben erhäten Sie eine alphabetische Liste.
Neuen Mitarbeitertypen hinzufügen
Klicken Sie hier um einen neuen Mitarbeitertyp anzulegen. Sie werden zur Mitarbeitertypprofilseite weitergeleitet, wo Sie Namen und Beschreibung eingeben sowie - falls gewünscht - weitere Attribute hinzufügen können.
Vorhandene Mitarbeitertypen
Klicken Sie Edit um das Profil eines Mitarbeitertyps zu ändern. Klicken Sie Protokoll um die Geschichte der Ereignisse anzusehen. Um einen Mitarbeitertyp zu löschen, setzen Sie einen Haken in die Löschen-Box, gehen Sie ans Ende der Seite, und klicken Sie auf den Knopf Gewählte Löschen.
