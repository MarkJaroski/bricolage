Hier werden Medientypen, auch MIME-Types genannt, erstellt und editiert.

Suche
Sucht Medientypen nach Namen. Eine leere Suche listet alle Medientypen auf, über die Buchstaben erhäten Sie eine alphabetische Liste.
Neuen Medientyp hinzufügen
Klicken Sie hier um einen neuen Medientyp anzulegen. Sie werden zur Medientypprofilseite weitergeleitet.
Vorhandene Medientypen
Klicken Sie Edit um das Profil eines Medientyps zu ändern. Klicken Sie Protokoll um die Geschichte der Ereignisse anzusehen. Um ein Medientyp zu löschen, setzen Sie einen Haken in die Löschen-Box, gehen Sie ans Ende der Seite, und klicken Sie auf den Knopf Gewählte Löschen.
