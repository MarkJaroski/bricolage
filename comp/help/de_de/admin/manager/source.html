Eine Quelle ist die Person oder Instanz, die für das zur Verfügung stellen von gewissen Inhaltsteilen verantwortlich ist. Partner Websites, Ihre eigene Site, freiberufliche Mitarbeiter oder Presseagenturen sind einige Beispiele möglicher Quellen. Bei Erstbenutztung werden Sie nur eine Quelle im System finden. Diese, vorübergehend "Interne Quelle" benannt, wird dazu verwendet, Inhalt zu Ihrer Site hinzuzufügen. Es ist empfehlenswert, diese Quelle mit dem Namen Ihrer Organisation umzubenennen und weitere Quellen, von denen Sie Inhaltsmaterial bekommen, hinzuzufügen. Wenn eine Quelle erstellt wurde, kann Sie benannt werden, indem sie im Profil eines Inhaltsteiles ausgewählt wird.

Suche
Sucht Quellen nach Namen. Eine leere Suche listet alle QUellen auf, über die Buchstaben erhäten Sie eine alphabetische Liste.
Neue Quelle hinzufügen
Klicken Sie hier um eine neue Quelle anzulegen. Sie werden zur Quellenprofilseite weitergeleitet.
Vorhandene Quellen
Klicken Sie Edit um das QUellenprofil zu ändern. Klicken Sie Protokoll um die Geschichte der Ereignisse anzusehen. Um eine Quelle zu löschen, setzen Sie einen Haken in die Löschen-Box, gehen Sie ans Ende der Seite, und klicken Sie auf den Knopf Gewählte Löschen.
