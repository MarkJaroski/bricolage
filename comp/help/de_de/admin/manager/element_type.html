Hier werden Elementtypen definiert und editiert. Elementtypen sind im Grunde nur Gruppierungen von Elementen, in denen gewisse Eigenschaften der typgleichen Elemente gesetzt werden.

Suche
Sucht Elementtypen nach Namen. Eine leere Suche listet alle Elementtypen auf.
Neuen Elementtyp hinzufügen
Klicken Sie hier um einen neuen Elementtyp anzulegen. Sie werden zur Elementtypprofilseite weitergeleitet.
Vorhandene Elementtypen
Klicken Sie Edit um das Profil eines Elementtyps zu ändern. Klicken Sie Protokoll um die Geschichte der Ereignisse anzusehen. Um ein Elementtyp zu löschen, setzen Sie einen Haken in die Löschen-Box, gehen Sie ans Ende der Seite, und klicken Sie auf den Knopf Gewählte Löschen.
