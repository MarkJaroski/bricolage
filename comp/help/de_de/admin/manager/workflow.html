Hier werden die Mananagementbereiche für Inhaltsbestandteile, die sog. Workflows (Arbeitsflüsse) erstellt und verwaltet. Die voreingestellten Workflows sind Medien, Beitrag und Vorlage, aber Sie können neue Workflows hinzufügen oder die bestehenden an die Bedürfnisse Ihrer Site anpassen.

Suche
Sucht Workflows nach Namen. Eine leere Suche listet alle Workflows auf, über die Buchstaben erhäten Sie eine alphabetische Liste.
Neuen Workflow hinzufügen
Klicken Sie hier um einen neuen Workflow anzulegen. Sie werden zur Workflowprofilseite weitergeleitet.
Vorhandene Workflows
Klicken Sie Edit um das Workflowprofil zu ändern. Klicken Sie Protokoll um die Geschichte der Ereignisse anzusehen. Um einen Workflow zu löschen, setzen Sie einen Haken in die Löschen-Box, gehen Sie ans Ende der Seite, und klicken Sie auf den Knopf Gewählte Löschen.
