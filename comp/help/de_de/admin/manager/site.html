Hier werden Sites erstellt und verwaltet.

Suche
Sucht aktive Sites nach Namen. Eine leere Suche listet alle Sites auf.
Neue Site hinzufügen
Klicken Sie hier um eine neue Site anzulegen. Sie werden zur Siteprofilseite weitergeleitet. Tragen Sie die benötigten Informationen ein und klicken Sie am Ende der Seite auf Speichern.
Vorhandene Sites
Klicken Sie Edit um das Profil der Site zu ändern. Klicken Sie Protokoll um die Geschichte der Ereignisse anzusehen. Um eine Site zu löschen, setzen Sie einen Haken in die Löschen-Box, gehen Sie ans Ende der Seite, und klicken Sie auf den Knopf Gewählte Löschen.
