Die Rechte einer Gruppe bestimmen die Privilegien seiner Mitglieder. In diesem Bereich können Sie neue Gruppen anlegen und die Rechte existierender ändern.

Suche
Sucht aktive Gruppen nach Namen. Eine leere Suche listet alle Gruppen auf, über die Buchstaben erhäten Sie eine alphabetische Liste. Sie können auch auch über das 'Typ auswählen' Menü einen Gruppentyp zum auflisten vorgeben.
Neue Gruppe hinzufügen
Klicken Sie hier um eine neue Gruppe anzulegen. Sie werden zur Gruppenprofilseite weitergeleitet, wo Sie den Gruppennamen und den gewünschten Gruppentyp angeben und wählen, welche Rechte sie zugewiesen bekommt.
Vorhandene Gruppen
Klicken Sie Edit um das Profil einer Gruppe zu ändern. Klicken Sie Protokoll um die Geschichte der Ereignisse anzusehen. Um eine Gruppe zu löschen, setzen Sie einen Haken in die Löschen-Box, gehen Sie ans Ende der Seite, und klicken Sie auf den Knopf Gewählte Löschen.
