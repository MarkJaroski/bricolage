Name
Tragen Sie den vollen Namen des Benutzers zusammen mit evtl. gewünschten Prä- und Suffixen ein.
Kontakte
Tragen Sie die Kontaktinformationen des Benutzer ein. Verwenden Sie die Pulldowns, um die Kontaktart (Telefon, Email, Fax, etc.) zu wählen und die Felder auf der rechten Seite, um die Daten einzutragen. Wenn Sie mehr als vier Felder benötigen, klicken Sie auf 'Weitere hinzufügen'.
Gruppenmitgliedschaften
Ermitteln Sie, welche Privilegien der Benutzer haben soll. Wählen Sie aus der Liste auf der linken Seite, zu welchen administrativen Gruppen der Benutzer hinzugrfügt werden soll. Wählen sie je eine und klicken Sie auf 'Zur Liste hinzufügen'. Die Liste auf der rechten Seite zeigt die Gruppen, denen der Benutzer aktuell zugehörig ist. Um eine zu entfernen, wählen Sie sie an und klicken Sie auf 'Von Liste entfernen'.
Benutzername und Paßwort
Tragen Sie die Anfangswerte für den Benutzernamen und das Paßwort des Anwenders ein. Der Anwender wird später diese Werte anpassen können.

Wenn Sie fertig sind, klicken Sie auf Speichern.
