在這你可以新增帳號跟設定存取權限．他確定使用者的存取定義或她是不是在管理群組 裡的會員．例如，假如你要Mary Jones允許編輯故事，你可以到她的使用者檔案網頁和把 她輸入到管理群組裡擁有編輯故事的權力．

搜尋
搜尋啟動的使用者．你可以用空白搜尋所有已啟動使用者
新增一個使用者
點選這按鈕就可以新一個新的使用者帳號．然後到使用者檔案網頁． 輸入所需的資訊何點選網頁的儲存鈕
已存在的使用者
點選Edit(編輯)修改人們的使用者檔案．點選Log(日誌)察看帳號的歷史動作．要刪除 使用者帳號，檢查是否點選Delete(刪除)核選方塊，然後把畫面捲動到最下面，並按下 Delete Checked（刪除已選取的帳號）的帳號按鈕．
